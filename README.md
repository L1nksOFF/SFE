# SFE
## 1. Назва продукту: Student File Exchange
## 2. Student File Exchange - це десктопний додаток на JavaFX, який  розроблений для спрощення процесу обміну навчальними матеріалами між студентами. Програма використовує Google Drive як хмарне сховище, що дозволяє користувачам безкоштовно завантажувати, зберігати та ділитися файлами.
## 3. Команда:
Бругош Владислав: 
Обов'язки: Планування архітектури проекту, розробка інтерфейсу користувача, написання коду для взаємодії з Google Drive API та локальною базою даних SQLite, тестування та подальша підтримка програми.
